Wat is mijn intresse in dit vak?,
Ik heb het altijd heel leuk gevonden om met technologie te werken. Ook heb ik altijd interesse in de nieuwste technologie in de markt en hoe het gemaakt wordt. Mijn grote motivatie in dit vak is om het volgende grote ding te maken dat ooit de wereld zal veranderen. Ik ben klaar om alles te doen om dit doel te bereiken. Voor mijn eerst opdracht ben ik een reserveringssysteem aan het maken met create functions voor Pidixy. Ik vind dit soort opdrachten heel leuk want ik vind het leuk om iemand blij te maken met mijn technologie. en ik kan dan ook een goede verandering maken voor dit bedrijf.


Voor mijn tweede project heb ik samen met een team Soundoff-Sports gemaakt. dit is een applicatie dat mensen met een auditieve beperking helpt met sporten. ik heb dit doel bereikt om een systeem te maken dat tekst bubbles stuurt van één pagina naar de andere. op deze andere pagina wordt deze tekst dan groot gedisplayed zodat mensen met een auditieve beperking beter instructies kunnen volgen van de instructeur. ik vind dit soort opdrachten heel leuk want ik kan mensen dan echt helpen met een beperking en de wereld van sport makkelijker voor hun maken.


Voor mijn derde en laatste project van jaar 1 heb ik een videogame gemaakt met een team van 4 genaamd museumroof. Dit is een game waar je speelt als politie of boef. Het doel van de politie is om kunstwerken te beschermen, En het doel van de boef is om zoveel mogelijk kunstwerken te stelen. Ik vond het werken aan deze game heel leuk om te doen want ik wil later graag in de game industrie werken. het is ook de reden waarom ik voor deze opleiding heb gekozen, om games te maken. het werken aan dit project vond ik het leukste van alle projecten. hopelijk maak ik later in de toekomst nog betere en leukere games. Ik wil later een grote game bouwen die heel veel mensen leuk gaan vinden.

Mijn CLE projecten

Project 1: Waterinator

Jaar 1: P01

Tijdens CLE1 hebben we een product moeten maken en presenteren. Dit product was de Waterinator. dit is een timer die je kan instellen op een half uur en een uur. en als deze timer afgaat betekend het dat je moet drinken. we hebben dit zo gemaakt omdat de persoon voor wie we dit hebben gemaakt vergeet te drinken tijdens het gamen.

Project 2 PHP basics systeem

Jaar 1: P02

Tijdens CLE2 heb ik een php systeem gemaakt voor een opdrachtgever. Hiervoor moesten we een CRUD maken, Create, Read, Update en Delete. met dit systeem kan je een agenda maken wat automatisch een detail pagina maakt waar je klanten op kunnen kijken.

Project 3 JavaScript web service

Jaar 1: P03

Tijdens CLE3 heb ik gewerkt in groepsverband aan een web service. Voor de web service heeft iedereen eeen individuele pagina moeten maken. Hierbij heb ik javascript toegepast voor mijn foto's en suggesties. voor dit project heb ik een functie aangemaakt om voor mensen met een auditieve beperking het makkelijker te maken om te sporten in groepsverband.

Project 4 Excalibur JS Game

Jaar 1: P04
